## Example Code

### Using Lists as Arrays

```python
# Define a list (like an array)
numbers = [1, 2, 3, 4, 5]

# Accessing elements
print(numbers[0])  # Output: 1
print(numbers[-1]) # Output: 5

# Modifying elements
numbers[2] = 10
print(numbers)  # Output: [1, 2, 10, 4, 5]

# Adding elements
numbers.append(6)
print(numbers)  # Output: [1, 2, 10, 4, 5, 6]

# Removing elements
numbers.remove(10)
print(numbers)  # Output: [1, 2, 4, 5, 6]
```
